Напишите программу, которая будет использовать анонимную функцию для фильтрации списка чисел. 
Функция должна принимать список чисел и возвращать новый список, содержащий только те числа, которые делятся на 3 без остатка.

Далее, используйте рекурсивную функцию для вычисления суммы всех чисел в полученном списке.

Используйте итератор для печати каждого числа в полученном списке.

Наконец, используйте генератор для создания нового списка, содержащего квадраты всех чисел в исходном списке.
