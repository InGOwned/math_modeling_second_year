Задача: "Города"

Описание: 
Есть несколько городов. Каждый город имеет свою специализацию, а также население, площадь и культуру. 
Город может:
- развиваться
- построить дом
- провести культурное мероприятие

Класс City:

1. Метод __init__(self, name, population, area, culture, specialization=specialization): Конструктор класса, который принимает название города, население, площадь, культуру и специализацию.
Если специализация не указана, то используется значение по умолчанию. Свойства name, population, area, culture следует сделать приватными

2. Использовать декоратор @property для получения значений свойств name, population, area, culture.

3. Метод develop(self): Метод, который выводит сообщение о развитии города.

4. Метод build(self): Метод, который выводит сообщение о постройке нового дома в городе.

5. cultural_event(self): Метод, который выводит сообщение о проведении культурного мероприятия в городе.

Подклассы DevelopedCity, BuildingCity, CulturalCity:

1. Метод __init__(self, name, population, area, culture): Конструкторы подклассов, которые вызывают конструктор базового класса City с указанием специализации.

2. Методы develop, build, cultural_event: Методы, которые переопределяют методы базового класса и выводят сообщения, характерные для каждого подкласса.

Класс Game:

__init__(self): Конструктор класса, который инициализирует пустой список городов.

add_city(self, city): Метод, который добавляет город в список городов.

play(self): Метод, который выбирает случайный город из списка городов, выводит информацию о нем и выполняет действия в зависимости от его специализации. 
Если список городов пуст, то выводит сообщение об этом.
